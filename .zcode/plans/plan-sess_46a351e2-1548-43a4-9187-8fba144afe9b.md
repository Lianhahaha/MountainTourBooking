## Goal
Redesign UI/UX to a mountain-like "Dusk Trail" aesthetic, spotlight the brand **LAAG TA ORB** on the landing page, and add a dedicated `/gallery` page (title + summary + location + date per photo). Light + dark themes retained. Database/Firebase + admin-upload UI explicitly **deferred** to a later pass — but the gallery data shape is built now so that pass is a clean plug-in. No backend changes; existing Supabase code left untouched.

## 1. Visual system — "Dusk Trail" (`src/app/globals.css`)
Replace the palette tokens with warm earth + ember values, keep the same token names so every component updates automatically:

**Light (`:root`)**
- `--background: #FBF6EE` (sand) · `--foreground: #241A14` (deep brown ink)
- `--surface: #FFFBF3` · `--surface-elevated: #F2E9D8`
- `--muted: #75655A` · `--accent: #C2410C` (ember) · `--accent-hover: #9A3412`

**Dark (`.dark`)**

## 4. New dedicated Gallery page
- **`src/data/gallery.ts`**: extend the data model so it's ready for the future admin/Firebase pass — each item gains `title`, `summary`, `date`. Backfill the 8 existing items with titles/summaries/dates (no new images needed).
